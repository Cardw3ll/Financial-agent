# Cardw3ll AI 💹

A local AI financial analyst built with Python, Flask and Ollama.

Upload any financial report (PDF, Excel, CSV) and ask questions about it. 
The AI reads the document and answers using only the figures provided — 
no hallucinations, no internet required, no data leaving your machine.

## Features
- 📄 Upload PDF, Excel, CSV or TXT financial reports
- 💬 Chat interface with streaming responses
- 📋 Quick actions — Summary, Ratios, Risks, Trends
- 🔒 100% local — your financial data never leaves your machine
- 🧠 Powered by Ollama — works with Mistral, DeepSeek, LLaMA and more

## Requirements
- Python 3.10+
- Ollama (ollama.ai)
- Any Ollama model (recommended: deepseek-r1:8b or higher)

## Setup
clone the repo, create a virtual environment, install requirements, run app.py
